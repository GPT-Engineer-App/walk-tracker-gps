# walk-tracker-gps

A mobile app that tracks the length and time of a walk using GPS and displays the route on Google Maps.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/walk-tracker-gps.git
cd walk-tracker-gps
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
